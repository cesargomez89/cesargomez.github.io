Github pages using Ghost + Buster

30 Jan 2016

on Github | Tools
Dependencies
  • Node JS ( 0.10.x, 0.12.x and 4.2.x only)
  • Python installer
  • wget
Ghost

Ghost is a blog template engine, you can pay for the hosting an other stuff they provide
or you can download the gost package and install it on your own server.

Since we want to install it on github we are going to download the package from: 

https://ghost.org/download/

Now inside the ghost folder run: 

npm install  
npm start

Now you can create your user and edit youe website visiting:

http://cesargomez89.github.io/ghost
or
http://cesargomez89.github.io/admin

You have a default theme called Casper, the themes are inside content/themes. There are a lot of free/paid ghost themes, just add the folder under themes and select it in the ghost general settings.

Buster

When you finished to customize your website locally, it's time to generate the static website using Buster.

To install buster just run: 

npm install buster

Just remember that Buster requires wget to run properly.
Create a new folder for your website 

mkdir my-website-buster

Go inside and run buster setup, buster will ask for the repo, just paste your github pages repo url git@github.com:cesargomez89/cesargomez89.github.io.git.

If you don't know how to use github pages go here

With your ghost server running, inside your my-website-buster, run the next command:

buster generate --domain=http://127.0.0.1:2368

Buster will generate the static website inside the static folder. Go into the static folder and commit everything and push it to the remote master branch. 

cd static  
git add .  
git commit -m 'first website'  
git push origin master -f

Wait like 10 minutes and that's it.
You have your own website running on github pages.
