Neovim Dotfiles <3

30 Jan 2016

on Vim | Dotfiles

As you maybe already know there is a new version of Vim called Neovim. The main difference between both is that Neovim works asynchronously.

I had been using a famous vim distribution made by Carlhuda Janus, but being reallistic, I don't use all the plugins that it contains so I was getting a little slow because the plugins excess. So I made this clean dotfiles built with the less plugins as possible and added some new async packages out there.

Neovim Dotfiles


Pre-requisites
  • curl
  • neovim
  • iterm
Install

The installation is really easy, just run this command in your term: 

curl -L https://raw.githubusercontent.com/cesargomez89/neovim-dotfiles/master/install.sh | bash

The script will keep a copy of your old vimrc
and rename it to .vimrc.old, so you don't care about lose your old config file.
