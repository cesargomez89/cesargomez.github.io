Ctags on vim

The Ctags help you to find any method declaration or the next place where is being used
even if the method is in another file.

http://ctags.sourceforge.net/

Install

brew install ctags

Implement

Ctags supports a lot of languages but since I'm a rails developer I just run this inside the project: 

  
ctags -R --languages=ruby,javascript --exclude=.git --exclude=log .

Here are the basics:

You can jump into the method with: 

  
:ta attr_accessor

or using regular expressions like that:

:ta /^before_*

Shortcuts

CTRL+] it will take you into the method.
CTRL-T will take you back from that method.

CTRL-I and CTRL-O will take you In and Out from the method.

Lists tags matching the given expression 

:ts [expr]

Jumps to the next or previous tag 

:[count]tn
:[count]tp

Jumps to the first or last matching tag 

:[count]tf  
:[count]tl
